```python
"""SQLite persistence layer for Steam Sniper.

Single source of truth: replaces watchlist.json.
Used by both Telegram bot (main.py) and future FastAPI dashboard.
"""
from __future__ import annotations

import json
import logging
from contextlib import contextmanager
from pathlib import Path
from typing import TYPE_CHECKING

from beartype import beartype

if TYPE_CHECKING:
    import sqlite3
    from collections.abc import Generator

logger = logging.getLogger("sniper.db")

DB_PATH = Path(__file__).parent / "data" / "sniper.db"
```

**Connection factory (`get_conn`):**
- `sqlite3.connect(str(DB_PATH), timeout=5.0)`
- `conn.row_factory = sqlite3.Row`
- Execute PRAGMAs: `journal_mode=WAL`, `busy_timeout=5000`, `foreign_keys=ON`
- Context manager: yield conn, commit on success, rollback on exception, close in finally

**`init_db()`:** Use `conn.executescript()` with CREATE TABLE IF NOT EXISTS for:
- `watchlist`: id INTEGER PK AUTOINCREMENT, name TEXT NOT NULL, name_lower TEXT NOT NULL, type TEXT NOT NULL CHECK(IN('buy','sell')), target_rub REAL NOT NULL, added_price_usd REAL, added_at TEXT NOT NULL, qty INTEGER DEFAULT 1
- UNIQUE INDEX `idx_watchlist_name_type` ON (name_lower, type)
- `price_history`: id INTEGER PK AUTOINCREMENT, name_lower TEXT NOT NULL, price_usd REAL NOT NULL, ts TEXT NOT NULL DEFAULT (datetime('now'))
- INDEX `idx_ph_name_ts` ON (name_lower, ts)
- `alerts`: id INTEGER PK AUTOINCREMENT, name TEXT NOT NULL, type TEXT NOT NULL CHECK(IN('buy','sell')), price_usd REAL NOT NULL, target_rub REAL NOT NULL, ts TEXT NOT NULL DEFAULT (datetime('now')), message TEXT
- `exchange_rates`: currency TEXT PRIMARY KEY, rate REAL NOT NULL, updated_at TEXT NOT NULL

**`get_watchlist()`:** SELECT * FROM watchlist ORDER BY added_at. Build and return `{"buy": [...], "sell": [...]}` where each entry is a dict from sqlite3.Row.

**`upsert_item(name, type_, target_rub, added_price_usd, added_at, qty=1)`:** INSERT INTO watchlist VALUES ON CONFLICT(name_lower, type) DO UPDATE SET target_rub, added_price_usd, added_at.

**`remove_item(name)`:** DELETE FROM watchlist WHERE name_lower=name.lower(). Return cur.rowcount.

**`get_watchlist_names()`:** SELECT DISTINCT name_lower FROM watchlist. Return as `set[str]`.

**`insert_price_snapshots(snapshots: list[tuple[str, float]])`:** executemany INSERT INTO price_history(name_lower, price_usd).

**`prune_old_history(days=90)`:** DELETE FROM price_history WHERE ts < datetime('now', '-{days} days'). Return rowcount.

**`log_alert(name, type_, price_usd, target_rub, message)`:** INSERT INTO alerts.

**`migrate_json_to_sqlite(json_path: Path, rate: float)`:** Read JSON, for each entry: if target > 100.0, treat as RUB (store as target_rub directly); if target <= 100.0, multiply by rate to get target_rub. Use upsert_item for idempotency. Default qty=1. Log each decision. Return count of migrated items.

**`get_cached_rate(currency="USD")`:** SELECT rate FROM exchange_rates WHERE currency=?. Return float or None.

**`save_rate(currency, rate)`:** INSERT INTO exchange_rates ON CONFLICT DO UPDATE.

All public functions decorated with `@beartype`. Use `from __future__ import annotations` at top.
