**1. `get_price_history(name: str, tf: str = "7d") -> list[dict]`**

Returns price history rows for charting. Each row: `{"price_usd": float, "ts": str}`.

Timeframe filtering logic:
- `"24h"` -> `WHERE ts >= datetime('now', '-1 day')`
- `"7d"` -> `WHERE ts >= datetime('now', '-7 days')`
- `"30d"` -> `WHERE ts >= datetime('now', '-30 days')`
- `"all"` -> no time filter

Query:
```sql
SELECT price_usd, ts FROM price_history
WHERE name_lower = ?
  AND ts >= datetime('now', ?)   -- omit for 'all'
ORDER BY ts ASC
```

Map tf to SQLite modifier: `{"24h": "-1 day", "7d": "-7 days", "30d": "-30 days"}`. If tf not in map and not "all", default to "7d".

Return `[dict(row) for row in rows]`.

**2. `get_portfolio_stats(prices: dict[str, float]) -> dict`**

Takes a dict of `{name_lower: price_usd}` (current prices from collector cache). Computes:
- `total_items`: count of watchlist entries
- `total_value_rub`: sum of `price_usd * rate * qty` for all items with known prices
- `total_added_rub`: sum of `added_price_usd * rate * qty` for items with added_price_usd > 0
- `delta_pct`: `((total_value_rub - total_added_rub) / total_added_rub) * 100` if total_added_rub > 0 else 0

Uses `get_watchlist()` internally + `get_cached_rate()` for USD/RUB conversion.

Returns:
```python
{
    "total_items": int,
    "total_value_rub": float,
    "total_added_rub": float,
    "delta_pct": float,
}
```

**3. `get_recent_alerts(limit: int = 20) -> list[dict]`**

For activity feed. Query:
```sql
SELECT name, type, price_usd, target_rub, ts, message
FROM alerts ORDER BY ts DESC LIMIT ?
```

Return `[dict(row) for row in rows]`.

**Lifespan (startup/shutdown):**
```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    db.init_db()
    await _collect_once()          # immediate first fetch
    _collector_task = asyncio.create_task(_collector_loop())
    yield
    if _collector_task:
        _collector_task.cancel()
```

**Collector loop (`_collector_loop`):**
- `while True`: sleep COLLECT_INTERVAL, then call `_collect_once()`
- Wrap in try/except to never crash the loop

**`_collect_once()`:**
- Fetch lis-skins JSON (use `httpx.AsyncClient` -- add `httpx>=0.28` to deps, or use `urllib` in `asyncio.to_thread`). Actually, use `urllib.request` in `asyncio.to_thread(fetch_prices_sync)` to stay consistent with main.py pattern and avoid adding httpx.
- Parse into `_prices` dict (same format as main.py: `{name_lower: {name, price, url, count}}`)
- Fetch USD/RUB from CBR, update `_usd_rub`, call `db.save_rate("USD", rate)`
- Snapshot prices for watched items: get `db.get_watchlist_names()`, filter `_prices`, call `db.insert_price_snapshots(snapshots)`
- Call `db.prune_old_history(90)`
- Log: `"Collected {len(_prices)} items, {len(snapshots)} snapshots"`

**Helper functions (sync, run via `asyncio.to_thread`):**
```python
def _fetch_lis_skins() -> list[dict]:
    """Fetch full lis-skins catalog. Returns raw list."""
    req = urllib.request.Request(LISSKINS_URL, headers={"User-Agent": "SteamSniper/1.0"})
    with urllib.request.urlopen(req, timeout=30) as resp:
        return json.loads(resp.read().decode("utf-8"))

def _fetch_usd_rub() -> float:
    """Fetch USD/RUB rate from CBR."""
    req = urllib.request.Request(CBR_URL, headers={"User-Agent": "SteamSniper/1.0"})
    with urllib.request.urlopen(req, timeout=10) as resp:
        data = json.loads(resp.read().decode("utf-8"))
    return float(data["Valute"]["USD"]["Value"])
```

**API Endpoints:**

**GET /api/watchlist** (API-02):
- Read `db.get_watchlist()` -> `{"buy": [...], "sell": [...]}`
- For each item, merge with `_prices` to add current price and delta:
  ```python
  item_data = _prices.get(entry["name_lower"])
  current_price_usd = item_data["price"] if item_data else None
  ```
- Compute delta_pct: `((current - added) / added) * 100` if added > 0
- Compute price_rub: `current_price_usd * _usd_rub`
- Return JSON:
  ```json
  {
    "buy": [
      {
        "name": "Kilowatt Case",
        "type": "buy",
        "target_rub": 65.0,
        "current_price_usd": 0.78,
        "current_price_rub": 63.5,
        "added_price_usd": 0.85,
        "delta_pct": -8.2,
        "qty": 1,
        "count": 1234,
        "url": "https://lis-skins.com/...",
        "added_at": "2026-04-12T..."
      }
    ],
    "sell": [...],
    "usd_rub": 83.45,
    "total_items_lis": 24000,
    "updated_at": "2026-04-12T18:30:00"
  }
  ```

**POST /api/watchlist** (API-03):
- Body (JSON): `{"name": str, "type": "buy"|"sell", "target_rub": float, "qty": int}`
- Look up current price from `_prices[name.lower()]` -> `added_price_usd`
- Call `db.upsert_item(name, type_, target_rub, added_price_usd, datetime.now().isoformat(), qty)`
- Return 201 with `{"ok": true, "name": name}`
- If name not found in `_prices`, still add but with `added_price_usd=0.0`

**DELETE /api/watchlist/{name}** (API-04):
- URL-decode name (FastAPI does this automatically for path params)
- Call `db.remove_item(name)` -> count
- If count == 0: return 404 `{"error": "not found"}`
- Else: return 200 `{"ok": true, "removed": count}`

**GET /api/search** (API-05):
- Query param: `q` (string, min 2 chars)
- Search `_prices` using same logic as `find_item()` in main.py: split query into words, filter items where all words appear in name_lower, sort by price, limit 12
- Return JSON:
  ```json
  {
    "results": [
      {"name": "...", "price": 0.78, "price_rub": 63.5, "count": 1234, "url": "..."}
    ],
    "query": "kilowatt"
  }
  ```

**GET /api/history/{name}** (API-06):
- Query param: `tf` (default "7d", allowed: "24h", "7d", "30d", "all")
- Call `db.get_price_history(name, tf)`
- Return JSON: `{"name": name, "tf": tf, "points": [{"price_usd": ..., "ts": ...}, ...]}`

**GET /api/stats** (API-07):
- Build prices dict from `_prices`: `{name_lower: price_usd}`
- Call `db.get_portfolio_stats(prices_map)`
- Also include: `usd_rub`, `total_lis_skins` (len of `_prices`), `last_update` timestamp
- Return JSON:
  ```json
  {
    "total_items": 5,
    "total_value_rub": 12500.0,
    "total_added_rub": 11000.0,
    "delta_pct": 13.6,
    "usd_rub": 83.45,
    "total_lis_skins": 24000,
    "last_update": "2026-04-12T18:30:00"
  }
  ```
