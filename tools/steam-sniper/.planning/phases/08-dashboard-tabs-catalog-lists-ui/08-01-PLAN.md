**Create static/js/catalog.js** -- Full catalog module with these functions:

```javascript
import { fmtRub } from './utils.js';
import { events } from './events.js';
```

Module-level state:
```javascript
let _category = '';      // '' means all
let _sort = 'name_asc';
let _query = '';
let _offset = 0;
const _limit = 50;
let _searchTimeout = null;
let _loaded = false;     // lazy load: only fetch when tab first activated
```

**`loadCatalog()`** -- async, exported:
1. `const params = new URLSearchParams({ limit: _limit, offset: _offset, sort: _sort })`
2. If `_category`: `params.set('category', _category)`
3. If `_query`: `params.set('q', _query)`
4. `const r = await fetch('/api/catalog?' + params)`
5. `const d = await r.json()`
6. Call `renderSidebar(d.categories, d.total)` -- only on first load or category change
7. Call `renderGrid(d.items)`
8. Call `renderPagination(d.total, d.offset, d.limit)`
9. Emit `events.emit('catalog:loaded', { total: d.total })` for cross-module use

**`renderSidebar(categories, total)`**:
1. Get `#catalogSidebar` element
2. Update `#catCountAll` with total
3. Build category buttons sorted by count descending. For each category:
   - Use display-friendly name from a `CAT_LABELS` const map:
     ```javascript
     const CAT_LABELS = {
       knife: 'Knives', gloves: 'Gloves', rifle: 'Rifles', pistol: 'Pistols',
       smg: 'SMGs', shotgun: 'Shotguns', machinegun: 'MGs', sticker: 'Stickers',
       case: 'Cases', graffiti: 'Graffiti', music_kit: 'Music Kits',
       patch: 'Patches', key: 'Keys', agent: 'Agents', other: 'Other'
     };
     ```
   - Create: `<button class="sidebar-item ${_category === cat ? 'active' : ''}" data-category="${cat}"><span class="sidebar-name">${label}</span><span class="sidebar-count">${count}</span></button>`
4. Preserve the "All" button at top, append category buttons below it
5. Set `active` class on current `_category` button ('' = All)

**`renderGrid(items)`**:
1. Get `#catalogGrid`
2. If items empty: show `<div class="empty-state">No items found</div>`
3. For each item, build card HTML:
   ```javascript
   const imgSrc = item.url
     ? 'https://community.akamai.steamstatic.com/economy/image/' + item.url + '/200fx200f'
     : '';
   const imgHtml = imgSrc
     ? `<img class="cat-card-img" src="${imgSrc}" loading="lazy" alt="">`
     : '<div class="cat-card-img-empty"></div>';
   const countCls = item.count > 0 ? '' : ' unavailable';

   `<div class="cat-card" data-item-name="${item.name}">
     <div class="cat-card-actions">
       <button class="list-toggle" data-list="favorite" data-name="${item.name}" title="Favorite">&#9825;</button>
       <button class="list-toggle" data-list="wishlist" data-name="${item.name}" title="Wishlist">&#9734;</button>
     </div>
     ${imgHtml}
     <div class="cat-card-name">${item.name}</div>
     <div class="cat-card-bottom">
       <div class="cat-card-price">${fmtRub(item.price_rub)}</div>
       <div class="cat-card-meta">
         <div class="cat-card-category">${item.category}</div>
         <div class="cat-card-count${countCls}">${item.count} qty</div>
       </div>
     </div>
   </div>`
   ```
4. Set grid innerHTML to joined card HTML

**`renderPagination(total, offset, limit)`**:
1. Get `#catalogPagination`
2. Calculate: `currentPage = Math.floor(offset / limit) + 1`, `totalPages = Math.ceil(total / limit)`
3. Build: Prev button (disabled if page 1) + page number buttons (show first, last, and 2 around current with ... gaps) + Next button (disabled if last page)
4. Show info: `"${offset + 1}-${Math.min(offset + limit, total)} of ${total}"`
5. Each page button: `<button class="cat-page-btn ${page === currentPage ? 'active' : ''}" data-page="${page}">${page}</button>`
6. Prev/Next: `<button class="cat-page-btn" data-page="${currentPage - 1}" ${currentPage <= 1 ? 'disabled' : ''}>&laquo;</button>`

**`initCatalog()`** -- exported:
1. Event delegation on `#catalogSidebar` for `.sidebar-item` clicks:
   - Get `data-category` value
   - Set `_category = value`, `_offset = 0`
   - Update active class on sidebar buttons
   - Call `loadCatalog()`
2. Event delegation on `#catalogPagination` for `.cat-page-btn` clicks:
   - Get `data-page` value (parseInt)
   - Set `_offset = (page - 1) * _limit`
   - Call `loadCatalog()`
   - Scroll catalog grid into view: `document.getElementById('catalogGrid').scrollIntoView({ behavior: 'smooth', block: 'start' })`
3. `#catalogSort` change listener:
   - Set `_sort = select.value`, `_offset = 0`
   - Call `loadCatalog()`
4. `#catalogSearch` input listener (debounced 400ms):
   - `clearTimeout(_searchTimeout)`
   - `_searchTimeout = setTimeout(() => { _query = input.value.trim(); _offset = 0; loadCatalog(); }, 400)`
5. Listen for tab activation to lazy-load:
   - `window.addEventListener('hashchange', () => { if (location.hash === '#catalog' && !_loaded) { _loaded = true; loadCatalog(); } })`
   - Also check on init: `if (location.hash === '#catalog') { _loaded = true; loadCatalog(); }`
