**4. Add chart JS functions** (add to the existing `<script>` section, at the end before the closing `</script>`):

Module-level variables:
```javascript
let chart = null;       // TradingView chart instance
let lineSeries = null;  // chart line series
let currentChartItem = null;  // name of item currently shown in chart
let currentTf = '7d';   // current timeframe
```

`async function showChart(itemName)`:
- Set `currentChartItem = itemName`
- Show `#chartPanel` (`style.display = ''`)
- Set `#chartItemName` text to itemName
- Highlight the clicked row: remove `.selected` from all `<tr>`, add `.selected` to the row with matching `data-item-name`
- If chart instance doesn't exist yet, create it:
  ```javascript
  chart = LightweightCharts.createChart(document.getElementById('chartContainer'), {
    layout: {
      background: { type: 'solid', color: '#111b2e' },
      textColor: '#94a3b8',
      fontFamily: "'JetBrains Mono', monospace",
      fontSize: 11,
    },
    grid: {
      vertLines: { color: 'rgba(26, 39, 68, 0.5)' },
      horzLines: { color: 'rgba(26, 39, 68, 0.5)' },
    },
    rightPriceScale: {
      borderColor: '#1a2744',
    },
    timeScale: {
      borderColor: '#1a2744',
      timeVisible: true,
    },
    crosshair: {
      vertLine: { color: 'rgba(255, 144, 106, 0.3)' },
      horzLine: { color: 'rgba(255, 144, 106, 0.3)' },
    },
  });
  lineSeries = chart.addSeries(LightweightCharts.LineSeries, {
    color: '#ff906a',
    lineWidth: 2,
    crosshairMarkerBackgroundColor: '#ff906a',
    priceFormat: { type: 'custom', formatter: (p) => Math.round(p).toLocaleString('ru-RU') + ' ₽' },
  });
  ```
- Call `await loadChart(itemName, currentTf)`

`async function loadChart(name, tf)`:
- Fetch `'/api/history/' + encodeURIComponent(name) + '?tf=' + tf`
- Parse JSON. points = response.points
- Convert points to TradingView format: each point becomes `{ time: Math.floor(new Date(point.ts + 'Z').getTime() / 1000), value: point.price_usd * usdRub }` (convert UTC string to Unix timestamp, USD to RUB)
- Sort by time ascending
- If points.length === 0: show message "No data for this timeframe" inside chartContainer (overlay text) and clear stats
- Else: `lineSeries.setData(tvData)` and `chart.timeScale().fitContent()`
- Update stats: compute min, max, avg from RUB values. Set `#csMin`, `#csMax`, `#csAvg` formatted as "X,XXX ₽". Set `#csCount` from the last watchlist entry's `count` field (store it when showing chart) or from the entry in the last loaded watchlist data.

`function hideChart()`:
- Hide `#chartPanel` (`style.display = 'none'`)
- Remove `.selected` from all rows
- Set `currentChartItem = null`

**5. Wire up event handlers:**

- **Row click**: In the existing loadWatchlist() function (from 03-01), after rendering tables, add event delegation on `#watchlistArea`:
  ```javascript
  document.getElementById('watchlistArea').addEventListener('click', (e) => {
    const row = e.target.closest('tr[data-item-name]');
    if (row && !e.target.closest('.wl-remove')) {
      showChart(row.dataset.itemName);
    }
  });
  ```
  IMPORTANT: Only add this listener ONCE (use a flag or check). Do NOT re-add on every loadWatchlist() call. Best approach: add the listener in `init()` function, not inside loadWatchlist().

- **Timeframe buttons**: Add click handler on `#chartTimeframes` (event delegation):
  ```javascript
  document.getElementById('chartTimeframes').addEventListener('click', async (e) => {
    const btn = e.target.closest('.tf-btn');
    if (!btn || !currentChartItem) return;
    currentTf = btn.dataset.tf;
    document.querySelectorAll('.tf-btn').forEach(b => b.classList.remove('active'));
    btn.classList.add('active');
    await loadChart(currentChartItem, currentTf);
  });
  ```

- **Close button**: `document.getElementById('chartClose').addEventListener('click', hideChart)`

- **Chart resize**: Handle window resize to keep chart responsive:
  ```javascript
  window.addEventListener('resize', () => {
    if (chart) {
      const container = document.getElementById('chartContainer');
      chart.resize(container.clientWidth, 300);
    }
  });
  ```

**Critical implementation details:**
- TradingView Lightweight Charts v5 uses `LightweightCharts.createChart()` (global from CDN)
- Series creation: `chart.addSeries(LightweightCharts.LineSeries, options)` -- v5 syntax
- Data format: `[{time: unixTimestamp, value: number}]` sorted ascending by time
- Timestamps from API are UTC strings like "2026-04-10 12:00:00" -- append 'Z' before parsing with `new Date()` to ensure UTC interpretation
- All chart prices in RUB (multiply price_usd * usdRub)
- Price formatter on right axis shows RUB: `Math.round(p).toLocaleString('ru-RU') + ' ₽'`
