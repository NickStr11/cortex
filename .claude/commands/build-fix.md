Диагностируй и почини ошибку билда или запуска.

## Инструкция

### Шаг 1 — Воспроизведи ошибку

Запусти билд/запуск проекта и собери вывод ошибки:

**Python:**
```bash
uv run python -m <module> 2>&1 | tail -30
```

**Node.js:**
```bash
npm run build 2>&1 | tail -30
```

### Шаг 2 — Определи причину

Проанализируй ошибку:
1. Какой файл и строка?
2. Тип ошибки (import, syntax, type, runtime)?
3. Есть ли missing dependencies?

Проверь:
- `uv sync` / `npm install` — зависимости установлены?
- Правильная версия Python/Node?
- Нет ли circular imports?

### Шаг 3 — Исправь

1. Примени минимальный фикс
2. НЕ рефактори вокруг — только фикс ошибки
3. Перезапусти билд

### Шаг 4 — Подтверди

Запусти билд повторно. Если ОК:

```
BUILD FIX REPORT
================
Error: [тип ошибки]
Root cause: [причина]
Fix: [что сделано]
Files changed: [список]
Build status: PASS ✓
```

Если билд опять падает — повтори с шага 1 (макс 3 итерации).
