Запусти Metrics — трекинг эффективности AI-агентов (Jules vs Codex).

## Инструкция

Выполни Python-скрипт для сбора метрик:

```bash
cd tools/metrics && uv run python main.py
```

Скрипт проанализирует последние PR и Issue через `gh api` и выведет:
- Success rate (мерж / создание)
- Время от создания Issue до мержа PR
- Строки кода (LOC) per task
- Разбивку по агентам

## Вывод

Скрипт выведет отчёт в формате Markdown.
Если данных недостаточно, попробуй проверить наличие лейблов `jules` или `codex` в Issues/PR.
