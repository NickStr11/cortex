### Два бага в deploy.py
1. **Windows Path → forward slashes**: `Path(remote_path).parent` на Windows возвращает backslashes → SFTP на Linux ломался. Замена на `rsplit("/", 1)[0]`. Был инфинит-луп в `_ensure_remote_dir` — сессия на Python зависала при загрузке static/
2. **Windows console cp1251**: apt/uv выводит `→` и прочий Unicode. `charmap can't encode` при print. Фикс: `sys.stdout.reconfigure(encoding="utf-8")` в начале скрипта
- Оба фикса остались в коде. Полезно для всех будущих deploy.py на Windows
