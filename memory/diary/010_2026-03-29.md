# Session 010 — 2026-03-29

## Что сделано

### VoiceType: whisper.cpp Vulkan GPU (главное достижение)
- **Проблема:** faster-whisper на CPU = ~4 сек на 5-10с аудио. SuperWhisper (платный аналог) делает за ~300ms
- **Исследование:** SuperWhisper использует whisper.cpp (C++) с GPU. Наш faster-whisper (Python) — только CPU
- **Решение:** заменили faster-whisper на whisper.cpp CLI с Vulkan на AMD RX 7800 XT
- **Путь к решению:**
  1. Скачали бинарник (август 2025) — segfault с beam>1, работает с beam=1
  2. whisper-server.exe крашился на Vulkan, откатились на CLI (subprocess)
  3. GPU retry + CPU fallback — ловит ~50% крашей
  4. Собрали свежий бинарник v1.8.4 (CMake + MinGW + Vulkan SDK)
  5. MinGW бинарник крашился (ggml_uncaught_exception assert)
  6. **GGML_NO_BACKTRACE=1** — решило проблему (рекомендация Codex)
  7. BUILD_SHARED_LIBS=OFF — монолитная сборка
- **Результат:** encode 300ms, total ~850ms, 0 крашей за 5 прогонов, AVX512 задетектился
- **Бенчмарк:** быстрее SuperWhisper на Windows. Пользователь подтвердил
