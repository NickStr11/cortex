## Наблюдения
- GSD отлично работает для greenfield: 4 фазы за один прогон, субагенты справились
- Но post-deploy итерации (фидбек → фикс → деплой) эффективнее без GSD — прямые правки
- Деплой через paramiko inline удобнее чем deploy.py (не нужен интерактивный пароль)
- Множитель lis-skins (1.034) — эмпирический, может дрифтовать со временем
- Steam Market API — rate limit на pagination (10 items per page), pagination 3 страницы покрывает ~30 items

## Решения
- Steam Sniper = отдельный git repo внутри tools/, не часть cortex git
- Lis-skins rate = CBR × 1.034 (hardcoded multiplier)
- Русский поиск через Steam Market API (не локальный словарь)
- Время сервера = MSK через datetime(timezone(+3))
